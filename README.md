# Deep Learning from Scratch

This repository is my personal journey of understanding deep learning by building everything from scratch — starting from basic neural networks to CNNs, RNNs, LSTMs, and finally Transformers.

The notes are shared to everyone! So check out if you want.

---

## Motivation

AI makes coding extremely easy, so why doing this? 

Understanding is still valuable and fun. Practice is the real understanding.

We aim to:

- Understand the core concepts of deep learning by implementing them from scratch.
- Rebuild key architectures on our own.
- Develop intuition for deep learning.
- Connect mathematical ideas with real implementations

---

## What This Repo Covers

This repository follows the evolution of deep learning from simple function approximation to modern architectures:

### 0. Foundations
- Linear Regression
- Logistic Regression
- Loss functions
- Gradient Descent

### 1. Neural Network Basics
- Linear layers
- Activation functions
- Multi-layer perceptron (MLP)
- Backpropagation

### 2. Convolutional Neural Networks (CNN)
- Convolution from scratch
- Local pattern extraction
- Image experiments

### 3. Recurrent Neural Networks (RNN)
- Sequence modeling
- Hidden states
- Gradient vanishing problem

### 4. Attention
- Scaled dot-product attention
- Q / K / V intuition
- Information routing

### 5. Transformer
- Multi-head attention
- Positional encoding
- Encoder architecture
